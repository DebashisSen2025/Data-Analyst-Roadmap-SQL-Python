## 📖 Learning Path (Recommended Order)

### **Week 1-2: SQL Basics**
1. Read `SQL/SQL_Roadmap.md`
2. Follow `SQL/01_SQL_Basics.sql`
3. Practice SELECT, WHERE, GROUP BY queries

### **Week 3-4: SQL Intermediate**
1. Master `SQL/02_SQL_Intermediate.sql`
2. Learn JOINS (LEFT, RIGHT, INNER, FULL)
3. Solve practice questions

### **Week 5: SQL Advanced**
1. Study Window Functions in `SQL/03_SQL_Advanced.sql`
2. Learn CTEs and complex queries
3. Solve top 50 interview questions

### **Week 6-7: Python Basics**
1. Start with `Python/01_Python_Basics.ipynb`
2. Practice variables, loops, conditions

### **Week 8-10: Data Manipulation**
1. Master pandas in `Python/02_Pandas_Numpy.ipynb`
2. Learn filtering, grouping, merging

### **Week 11-12: Real Projects**
1. Pick a project from `Projects/`
2. Complete end-to-end analysis
3. Add to your portfolio

## 📊 Key Concepts You'll Master

| Concept | SQL | Python |
|---------|-----|--------|
| **Data Selection** | SELECT | filter(), loc[] |
| **Filtering** | WHERE | query() |
| **Grouping** | GROUP BY | groupby() |
| **Joining** | JOIN | merge() |
| **Aggregation** | COUNT, SUM, AVG | agg() |
| **Sorting** | ORDER BY | sort_values() |
| **Complex Logic** | CASE WHEN | apply() |
